# CondService

Aplicativo Android desenvolvido em Kotlin como projeto acadêmico da ETEC, com o objetivo de simular um sistema de gestão e comunicação para condomínios.

## Sobre o Projeto

O CondService foi criado para auxiliar moradores e administradores de condomínios em tarefas do dia a dia, oferecendo funcionalidades como reservas, autorizações, consulta de informações e comunicação com a portaria.

## Funcionalidades

* Login de usuários
* Cadastro de usuários
* Recuperação de senha
* Consulta de boletos
* Reserva de áreas comuns
* Cadastro de autorizações para visitantes
* Mural de avisos
* Prestação de contas
* Solicitação de mudança
* Contato com a portaria

## Tecnologias Utilizadas

* Kotlin
* Android Studio
* XML
* ConstraintLayout
* Intent
* Android SDK

## Conceitos Aplicados

Durante o desenvolvimento foram utilizados conceitos importantes de programação Android, como:

* Criação de interfaces com XML
* Navegação entre telas (Activities)
* Envio e recebimento de dados com Intent
* Validação de campos
* Manipulação de eventos de clique
* Uso de Toast para mensagens ao usuário
* Organização de layouts utilizando ConstraintLayout

## Estrutura do Projeto

* MainActivity – Tela de login.
* CadastroActivity – Cadastro de usuários.
* LoginActivity – Tela principal do sistema.
* EsqueceuActivity – Recuperação de senha.
* ReservaActivity – Reserva de espaços comuns.
* AutorizacaoActivity – Cadastro de visitantes autorizados.

## Como Executar

1. Clone este repositório.
2. Abra o projeto no Android Studio.
3. Aguarde a sincronização do Gradle.
4. Execute o aplicativo em um dispositivo Android ou emulador.

## Objetivo Acadêmico

Este projeto foi desenvolvido como atividade prática da ETEC, visando aplicar conhecimentos de desenvolvimento mobile com Kotlin e Android Studio.
